How to run this testplugin

Load the test project (TestStratosTestable.Stratos.Plugin) 
Build TestStratosTestable.Stratos.Plugin
Copy TestStratosTestable.Stratos.Plugin.dll to <your path>\Stratos\src\bin\Debug
Copy the Content folder <your path>\Stratos\src\TestStratosTestable.Stratos.Plugin\bin\Debug\Content to \Stratos\src\bin\Debug
F5
Go to http://localhost:1337/api/testPlugin/ for api
Go to http://localhost:1337/testPlugin/ for html view
